day01--总结

1、   java是面向对象的语言
        javaSE是基础，javaME是小型化语言（老年机应用程序）javaEE是企业化语言（网站开发）

2、  JDK是java语言的软件开发工具包（编译工具和运行工具）
       JRE是java语言运行时的环境（JVM和java核心库）
             （  JDK{ JRE：[ JVM  or  核心类库 ]  or  开发工具 } ）

3、  java程序可以跨平台，JVM不能跨平台

4、基本信息
{
      （1）关键字： 字母全部小写，编程软件上有提示
      （2）常量：    不会发生改变的量
      （3）变量：    在存储空间中声明一个变量，变量的值会改变，但是变量的名称不会
                变量使用时的注意事项：   变量名不能重复定义
                                                       一条语句可以定义多个变量名但是中间要用“ ，”隔开
                                                       变量在使用的时候要进行赋值
                                                       在使用float和long类型是要注意标识（大小写都可以“F”or“f”）
                                                       在哪个大括号内定义的变量只能在大括号内有效
}
数据类型：{
    基本数据类型数值型：{
           数值型：        【整数型（short：短/long：长/byte：字节），浮点型（float：单/double：双），
                                  字符型（char）】   
           非数值型：    【布尔（Boolean：true/false）】
    }
         
    引用数据类型：【类（class），接口（interface），数组（“[  ]”）】
}

5、键盘录入
       引包：（ import  java.util.Scanner ）
       对象的声明：   
                   Scanner sc = new Scanner(System.in);
       使用变量接收sc：
                   int  i  =  sc.nextInt();
                （用来接收的变量类型可以改变，后面的next~也要改变）

6、标识符
        组成：字母，数字，下划线，美元符号
        定义规则：
                开头不能是数字
                不能是关键字
                区分大小写
