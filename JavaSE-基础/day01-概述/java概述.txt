 java语言背景介绍
  
    java语言
        面向对象

    java语言的三个版本
       javaSE   javaME   javaEE
     javaSE ：java语言的基础， 用于桌面应用的开发，是其他的两个版本的基础

     桌面应用：用户只要打开程序，就能在最短时间内找到需要的功能

     javaME：java语言（小型版），用于嵌入式消费类的电子设备
     javaEE：java语言的（企业版），用于Web方向的网站开发
    
     网页： 通过浏览器将数据展示在用户面前，【和后台服务器没有交互】

     网站：通过跟后台服务器的交互，将查询到的真实数据再通过网页展示出来
                 （网站 = 网页 + 后台服务器）


 JRE和JDK
    开发java程序：（编写代码，编译代码，运行代码）
 
   JRE ：java运行时环境，包含JVM虚拟机及java核心类库

{
类：java文件在代码中的集合体现 （类 = java文件）
       类库： 存放多个java文件的仓库
       
  【我们在自己编写代码的过程中，需要用到java存放在JRE中，已经写好的java文件】       
}

    JDK：java语言的软件开发工具包，内部包含了代码的【编译工具和运行工具】

  JDK{JRE [ JVM  and  核心类库 ]  and  开发工具}
